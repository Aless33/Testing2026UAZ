
### ¿Que se tiene que hacer?

Una calculadora funcional para empezar a probar las distintas maneras de realizar una prueba funcional en Python
### ¿Como se puede hacer?

Con un archivo de pruebas y el código, mediante el de pruebas vamos a ir probando cada función de la calculadora, ya sea restas, sumas, divisiones, multiplicaciones, potencia, raíz, etc.
### ¿Que se hizo?

Una calculadora funcional separada por métodos dependiendo de la función que utiliza un archivo de pruebas en donde se prueba cada una de estas funciones y métodos.
### ¿Que se aprendió?

Las pruebas son medios necesarios para obtener una buena calidad de un programa, que no tenga defectos graves y que pueda ser capaz de probarse de distintas maneras.
